# 背单词小助手 - PWA英语学习应用

一个为儿童设计的PWA（渐进式Web应用）背单词应用，支持离线使用、智能复习和记忆曲线算法。

## 功能特点

### 核心功能
- 📚 **单词管理** - 添加、编辑、删除单词，支持英文、中文解释和例句
- 🎯 **智能复习** - 基于艾宾浩斯遗忘曲线的智能复习算法
- 📱 **PWA支持** - 可安装到手机主屏幕，支持离线使用
- 📊 **学习统计** - 追踪学习进度、正确率和连续打卡天数
- 🔄 **GitHub同步** - 与家人共享学习数据，多设备同步
- 🎨 **儿童友好界面** - 大字体、明亮色彩、简单交互

### 记忆曲线系统
采用简化版艾宾浩斯遗忘曲线：
- 第1次复习: 1天后
- 第2次复习: 2天后
- 第3次复习: 4天后
- 第4次复习: 7天后
- 第5次复习: 15天后

## 使用说明

### 学习模式
1. 打开应用后默认进入学习模式
2. 显示英文单词，点击卡片查看中文和例句
3. 选择"认识"或"不认识"进行标记
4. 系统会根据记忆曲线自动安排复习

### 管理单词
1. 点击底部"管理"标签
2. 点击"添加单词"输入新单词
3. 点击单词卡片可编辑或删除
4. 支持搜索和批量导入/导出

### 查看统计
1. 点击底部"统计"标签
2. 查看学习进度和成就
3. 了解每日学习趋势

### 个性化设置
1. 点击底部"设置"标签
2. 调整每日学习目标（5-100个单词）
3. 选择复习间隔模式（标准/快速/缓慢）
4. 调整字体大小和界面主题

## 数据管理

### GitHub同步（推荐）⭐
支持使用GitHub作为云存储，与家人共享学习数据，保持多设备学习进度同步。

#### 第一步：创建 GitHub Token

1. 访问 https://github.com/settings/tokens/new
2. 填写Token名称：`背单词同步`
3. 选择期限：`No expiration`（不过期）
4. 勾选权限：只需勾选 ✅ **repo**
5. 点击"Generate token"生成
6. **重要**：复制生成的Token（以`ghp_`开头），此Token只显示一次

#### 第二步：在应用中配置

1. 打开应用 → 点击底部"⚙️ 设置"
2. 找到"GitHub同步设置"部分
3. 填写以下信息：
   - Personal Access Token: 粘贴刚才的Token
   - GitHub用户名: 你的用户名
   - 仓库名称: `bdc`
   - 分支名称: `main`
4. 点击"测试连接"确认配置正确
5. 点击"保存配置"

#### 第三步：开始同步

**首次使用：**
- 手机A：添加单词 → "📝管理" → "☁️上传"
- 手机B：配置完成后 → "📝管理" → "📥下载"

**日常使用（推荐）：**
- 学习完成后点击 "📝管理" → "🔄同步"
- 自动合并双方学习进度，不会丢失数据

#### 三种同步方式

- 🔄 **智能同步**: 自动合并本地和远程数据，保留复习次数更多的版本（推荐）
- ☁️ **上传**: 将本地数据上传到GitHub（会覆盖远程数据）
- 📥 **下载**: 从GitHub下载数据到本地（会覆盖本地数据）

#### 常见问题

**Q: Token安全吗？**  
A: Token只保存在浏览器本地存储中，不会上传到其他服务器。但不要分享给他人。

**Q: 会不会覆盖对方的学习记录？**  
A: 使用"智能同步"不会！它会自动合并。只有"上传"和"下载"才会覆盖数据。

**Q: 遇到"连接失败"怎么办？**  
A: 检查Token是否正确、是否有`repo`权限、用户名和仓库名是否准确。

**Q: 数据存储在哪里？**  
A: 存储在你的GitHub仓库的`data/vocabulary-data.json`文件中，可以直接在GitHub上查看。

### 导入单词
支持JSON格式批量导入，格式如下：
```json
[
  {
    "english": "apple",
    "chinese": "苹果",
    "example": "I eat an apple every day."
  },
  {
    "english": "book",
    "chinese": "书",
    "example": "She is reading a book."
  }
]
```

### 导出备份
点击"导出"按钮可将所有单词导出为JSON格式，用于备份或分享。

## 技术架构

- **前端框架**: 原生JavaScript + HTML5 + CSS3
- **数据存储**: IndexedDB（本地存储）
- **云同步**: GitHub API + Personal Access Token
- **离线支持**: Service Worker
- **部署平台**: GitHub Pages

## 本地开发

1. 克隆仓库
```bash

cd bdc
```

2. 启动本地服务器
```bash
python -m http.server 8000
# 或使用其他静态服务器
```

3. 访问 `http://localhost:8000`
